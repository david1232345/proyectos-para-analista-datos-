# amazon-sales-analysis-2025/
Este es un proyecto de análisis de datos sobre las ventas de Amazon durante el año 2025.
Utilicé SQL para realizar consultas que permitieron obtener métricas específicas como las ventas netas, los productos más vendidos y los métodos de pago más usados.
Posteriormente, visualicé los resultados en Power BI para presentar la información de manera clara, dinámica y atractiva para la toma de decisiones.
